## Which source answers which question

The assistant is wired to three sources. It picks based on what you ask:

| Your question | Source |
| --- | --- |
| "How does X work / what's the process?" | **This KB** (Mintlify) |
| "What's the current status / count of record Y?" | **egora-data** (live, PII-redacted) |
| "What's the latest draft / not-yet-official?" | **Notion** |

## What the assistant will and won't do

<Check>It drafts customer replies and offer text (German, informal "Du", with placeholders).</Check>
<Check>It looks up live request/order status and counts via the connector.</Check>
<Check>It walks you through the lifecycle and the post-booking checklist.</Check>

<Warning>
  It will **not** invent prices, dates, quantities, statuses or IDs, and it does **not** take binding
  actions (sending offers, confirming bookings, invoicing, refunds) — those stay in the eGora backend,
  with you.
</Warning>
