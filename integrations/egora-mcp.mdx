The **egora-data MCP** connector gives Claude read-only, PII-redacted access to live eGora business data
— orders, catering requests, companies, KPIs and more.

<Note>
  This is **different** from the KB MCP. The KB MCP serves *knowledge* (how things work); egora-data
  serves *current data* (the actual numbers).
</Note>

## Setup

The full step-by-step (Claude Desktop, token, staging vs production, troubleshooting) lives in the IT
guide:

<Card title="Connect Claude to eGora data (MCP)" icon="plug" href="/it/mcp-setup">
  On Claude web (Cowork / Teams): Add custom connector → `https://egora-catering.de/mcp` → leave Client
  ID/Secret blank → Connect → authorize in Nova. (Token method available for Claude Code / Desktop.)
</Card>

## When to use it

Use egora-data when the answer depends on **current numbers**, e.g.:

- "How many B2B catering requests in the last 7 days, by city?"
- "What's our conversion rate this month?" (see also the [KPI catalog](/it/kpi-catalog))

For "how does X work" questions, use the KB instead.
