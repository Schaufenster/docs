Der **egora-data MCP**-Connector gibt Claude Lesezugriff auf Live-Geschäftsdaten von eGora — mit PII-Redaktion —
also Bestellungen, Catering-Anfragen, Firmen, KPIs und mehr.

<Note>
  Das ist **etwas anderes** als die KB MCP. Die KB MCP liefert *Wissen* (wie Dinge funktionieren); egora-data
  liefert *aktuelle Daten* (die tatsächlichen Zahlen).
</Note>

## Setup

Die vollständige Schritt-für-Schritt-Anleitung (Claude Desktop, Token, Staging vs. Produktion, Troubleshooting) findest du im
IT-Guide:

<Card title="Claude mit eGora-Daten verbinden (MCP)" icon="plug" href="/it/mcp-setup">
  Im Claude Web (Cowork / Teams): Custom Connector hinzufügen → `https://egora-catering.de/mcp` → Client
  ID/Secret leer lassen → Connect → in Nova autorisieren. (Token-Methode für Claude Code / Desktop verfügbar.)
</Card>

## Wann du ihn nutzt

Nutze egora-data, wenn die Antwort von **aktuellen Zahlen** abhängt, z. B.:

- "Wie viele B2B-Catering-Anfragen in den letzten 7 Tagen, nach Stadt?"
- "Wie ist unsere Conversion Rate diesen Monat?" (siehe auch den [KPI-Katalog](/it/kpi-catalog))

Für "Wie funktioniert X"-Fragen nutze stattdessen die KB.
