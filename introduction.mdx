## Wie sie aktuell bleibt

<Steps>
  <Step title="Versioniert in Git">
    Jede Änderung ist ein prüfbarer Pull Request in `Schaufenster/docs` – nichts ändert sich stillschweigend.
  </Step>
  <Step title="Ausgelieferte Features erzeugen automatisch einen Changelog-Entwurf">
    Mintlifys Changelog-Automatisierung entwirft einen Eintrag, wenn sich Produktcode ändert – siehe
    [Release-Kommunikation](/it/feature-changelog).
  </Step>
  <Step title="Claude liest immer den aktuellen Stand">
    Weil dieselbe Quelle Claude via MCP versorgt, antwortet der Assistent stets auf Basis des aktuellen Wissens.
  </Step>
</Steps>
