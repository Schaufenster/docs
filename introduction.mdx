## How it stays current

<Steps>
  <Step title="Versioned in git">
    Every change is a reviewable pull request in `Schaufenster/docs` — nothing changes silently.
  </Step>
  <Step title="Shipped features auto-draft a changelog">
    Mintlify's changelog automation drafts an entry when product code changes — see
    [Release communication](/it/feature-changelog).
  </Step>
  <Step title="Claude reads the latest">
    Because the same source feeds Claude via MCP, the assistant answers from current knowledge.
  </Step>
</Steps>
