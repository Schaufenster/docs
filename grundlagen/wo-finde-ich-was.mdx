> Schnelle Nachschlage-Tabelle: Welches Objekt / welche Info liegt in welchem System — und wie es dort heißt. Die Brücke zwischen Begriff und „wo klick ich". Begriffe selbst → [glossar.md](/grundlagen/glossar), Tool-Übersicht → [tools.md](/grundlagen/tools).

## Admin Backend (Laravel) — der operative Kern

| Was du suchst | Wo / wie es heißt |
|---|---|
| Caterer (als Betrieb) + ihre Produktkataloge | **Retailer** |
| Eingehende Anfragen | **Requests** |
| Bestellungen / ausgeführte Orders | **Bestellungen** |
| Angebote | **Offers / Angebote** |
| Einzelne Produkte | **Products** — oder über den jeweiligen **Retailer** in dessen Produktkatalog |
| Pakete | **Pakete** |
| Caterer-Standorte | **Filialen** |
| Rechnung erstellen | über die **Order** im Backend erstellen (Zahlungsabwicklung dann über Stripe) |
| AI-Features (Allergen-Tagging, Menu Creation, PDF-to-Offer) | im Admin Backend |

## Kunden- & Caterer-Sicht

| Was du suchst | Wo |
|---|---|
